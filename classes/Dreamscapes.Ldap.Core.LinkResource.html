    \Dreamscapes\Ldap\Core\LinkResource

    Search LDAP tree

    search(string $baseDn, string $filter, array $attributes, string $scope = self::SCOPE_SUBTREE, boolean $attrsOnly = false, integer $sizeLimit = 0, integer $timeLimit = 0, integer $deref = LDAP_DEREF_NEVER) : \Dreamscapes\Ldap\Core\ResultResource

    This method deviates from the native PHP implementation - there is only one method to perform search, list and read operations. The scope of the operation is controlled by the $scope parameter, which can be one of:

    self::SCOPE_SUBTREE - equivalent of ldap_search() ( default ) self::SCOPE_ONELEVEL - equivalent of ldap_list() self::SCOPE_BASE - equivalent of ldap_read()

    inherited_from \Dreamscapes\Ldap\LinkResourceInterface::search()

    Parameters

    $baseDn

    string

    The base DN for the directory

    $filter

    string

    Ldap query filter ( an empty filter is not allowed )

    $attributes

    array

    An array of the required attributes, e.g. array("mail", "sn", "cn")

    $scope

    string

    One of self::SCOPE_SUBTREE, self::SCOPE_ONELEVEL or self::SCOPE_BASE

    $attrsOnly

    boolean

    Should be set to 1 if only attribute types are wanted

    $sizeLimit

    integer

    Enables you to limit the count of entries fetched. Setting this to 0 means no limit

    $timeLimit

    integer

    Sets the number of seconds how long is spend on the search. Setting this to 0 means no limit.

    $deref

    integer

    Specifies how aliases should be handled during the search

    Returns
